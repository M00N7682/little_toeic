## 서비스 개요
본 웹사이트는 로그인 없이 매일 새로운 토익 문제를 제공하는 서비스임.
사용자는 접속 즉시 오늘의 RC 문제를 풀 수 있으며, 풀이 후 정답 및 해설을 확인할 수 있음.
모바일 사용자 중심으로 최적화된 UI를 제공함.

## 목표
- 로그인 없이 즉시 학습 가능
- 문제·정답·해설이 한 화면에서 확인 가능

## 타겟 사용자
- 토익 학습 대학생
- 취준생 및 직장인
- 하루 5~10문제 풀이 사용자

## 핵심 가치
- 로그인 없음
- 랜덤 문제 제공
- 모바일 친화 UI

## 페이지 구성
1. 메인(오늘의 문제)
2. 풀이 결과 페이지
3. About 페이지
