## 공통 규칙
- React 파일명: PascalCase
- Python 파일명: snake_case
- indent: React 2 spaces / Python 4 spaces
- 함수는 동사 기반 이름

## React 규칙
- 함수형 컴포넌트 사용
- 상태 관리: useState
- API 호출은 /utils/api.ts 중앙집중
- Tailwind 사용

## FastAPI 규칙
- 라우터는 /app/api 분리
- JSON 접근은 service 레이어
- 응답 구조 통일(success/data/error)
- pydantic 모델 필수
